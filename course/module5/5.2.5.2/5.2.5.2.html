Структура каталогов и атрибуты файлов

Местоположения файлов пользователя

По умолчанию в ОС Windows 7 и Windows Vista большинство файлов, созданных пользователями, хранится в папке C:\Users\Имя_пользователя\. В ОС Windows XP используется папка C:\Documents and Settings\Имя_пользователя\.

Кроме того, каждая папка пользователя содержит папки, предназначенные для хранения музыки, видео, информации с веб-сайтов, изображений, как показано на рис. 1. В этих папках также хранятся данные пользователей, необходимые для работы различных программ. Если на одном компьютере зарегистрировано несколько пользователей, каждый из них имеет свою папку с избранным, элементами рабочего стола и файлами cookie. «Cookie» — это файлы, содержащие информацию о посещенных пользователем веб-страницах.

Системная папка

После установки ОС Windows все файлы, используемые для работы компьютера, хранятся в папке C:\Windows\system32. Содержимое системной папки Windows показано на рис. 2.

Шрифты

В папке C:\Windows\Fonts находятся шрифты, установленные на компьютере. Шрифты имеют несколько форматов, среди них TrueType, OpenType, Composite и PostScript. Вот несколько примеров гарнитуры шрифтов: Arial, Times New Roman, Courier. Доступ к папке «Шрифты» осуществляется с панели управления. Шрифты устанавливаются при помощи меню Файл > Установить шрифт.

Временные файлы

В папке «Временные файлы» содержатся файлы, созданные ОС и программами на короткий промежуток времени. Например, временные файлы могут быть созданы во время установки приложения, чтобы для других приложений оставалось больше ОЗУ.

Временные файлы используются почти во всех программах. Они автоматически удаляются по окончании их использования ОС или приложением. Однако некоторые временные файлы необходимо удалять вручную. Поскольку временные файлы занимают место на жестком диске, которое могло бы использоваться другими файлами, рекомендуется удалять временные файлы каждые два-три месяца по мере необходимости.

В ОС Windows Vista и Windows 7 временные файлы обычно размещаются в следующих папках:

  • C:\Windows\Temp

  • C:\Users\Имя_пользователя\AppData\Local\Temp

  • %USERPROFILE%\AppData\Local\Temp

В ОС Windows XP временные файлы обычно размещаются в следующих папках:

  • C:\Temp

  • C:\Tmp

  • C:\Windows\Temp

  • C:\Windows\Tmp

  • C:\Documents and Settings\%USERPROFILE%\Local Settings\Temp

ПРИМЕЧАНИЕ. %USERPROFILE% является переменной среды, установленной ОС, с именем пользователя, который в данный момент выполнил вход в систему. Переменные среды используются операционной системой, приложениями и программами установки ПО.

Для отображения переменных среды в Windows 7 перейдите по следующим пунктам:

Пуск > Панель управления > Система > Дополнительные параметры системы > Дополнительно > Переменные среды

Для отображения переменных среды в Windows Vista перейдите по следующим пунктам:

Пуск > Панель управления > Система > Дополнительные параметры системы > Дополнительно > Переменные среды

Для отображения переменных сред в Windows XP перейдите по следующим пунктам:

Пуск > Панель управления > Система > Дополнительно > Переменные среды

Program Files

Папка "Program Files" используется большинством установочных программ для установки ПО. В 32-разрядных системах программы обычно устанавливаются в папку C:\Program Files. В 64-разрядных системах 64-разрядные программы обычно устанавливаются в папку C:\Program Files, а 32-разрядные программы — в папку C:\Program Files (x86).
