Ժամանակակից օպերացիոն համակարգեր

ՕՀ-ների թարմացում

Համակարգչի թարմացման գործընթացը կարող է ավելի արագ կատարվել քան նոր ՕՀ-ի տեղադրումը: ՕՀ-ի տարբերակից կախված թարմացման գործընթացը տարբեր է: Օրինակի համար Windows 7-ի տեղադրումը իրականացնող ծրագիրը (Setup Utility) փոխարինում է Windows Vista-ի ֆայլերը Windows 7-ի ֆայլերով՝ պահպանելով տեղադրված ծրագրերը և կարգավորումները:

Доступные варианты обновления зависят от версии операционной системы. Например, 32-разрядная ОС не может быть обновлена до 64-разрядной ОС. Еще один пример: Windows XP невозможно обновить до Windows 7. Прежде чем приступить к обновлению, изучите список возможных способов его выполнения на веб-сайте разработчика ОС.

Հիշեցում. Նախքան թարմացման գործընթացին անցնելը, հարկավոր է իրականացնել բոլոր տեսակի տվյալների պահուստավորում՝ տեղադրման հետ կապված խնդիրներից ապահովվագրված լինելու համար:

Որպեսզի թարմացնենք ՕՀ-ն մինչև Windows 7 հարկավոր է իրականացնել հետևյալ քայլերի հաջորդականությունը:

Քայլ 1. Տեղադրեք Windows 7-ի սկավառակը օպտիկական սկավառակատարի մեջ, կհայտնվի կարգավորումների պատուհանը

Քայլ 2. 2. Ընտրեք տարբերակը«Установить сейчас»(տեղադրել հիմա).

Քայլ 3. 3. Ձեզ կառաջարկվի բեռնել տեղադրման համար կարևոր բոլոր նորացումները։

Քայլ 4. Ընդունեք օգտագործողի լիցենզիոն համաձայնագիրը (End User License Agreement - EULA) և սեղմեք«Далее»(հաջորդը).

Քայլ 5. Սեղմեք«Обновить»(թարմացնել). Համակարգը կսկսի պատճենել տեղադրման ֆայլը։

Քայլ 6. Հետևեք հուշումներին՝ նորացումը ավարտին հասցնելու համար: Տեղադրման ավարտից հետո համակարգիչը կվերաբեռնավորվի:

Հիշեցում։ Նախքան նորացման գործընթացին անցնելը`windows XP-ից windows vista, պետք է տեղադրել Windows XP Service Pack 2 կամ 3:

Որպեսզի նորացնենք ՕՀ-ն մինչև Windows Vista հարկավոր է իրականացնել հետևյալ քայլերի հաջորդականությունը:

Քայլ 1. Տեղադրեք Windows Vista-ի սկավառակը օպտիկական սկավառակատարի մեջ, կհայտնվի կարգավորումների պատուհանը։

Քայլ 2. նտրեք տարբերակը «Установить Windows Vista»(տեղադրել Windows Vista)։

Քայլ 3. Ձեզ կառաջարկվի բեռնել տեղադրման համար կարևոր բոլոր նորացումները։

Քայլ 4. Մուտքագրեք արտադրանքի բանալին և ընդունեք օգտագործողի լիցենզիոն համաձայնագիրը (EULA)։

Քայլ 5. Սեղմեք «Обновить»(նորացնել). Համակարգը կսկսի պատճենել տեղադրման ֆայլը։

Քայլ 6. Հետևեք հուշումներին՝ նորացումը ավարտին հասցնելու համար: Տեղադրման ավարտից հետո համակարգիչը կվերաբեռնավորվի:

Որպեսզի նորացնենք ՕՀ-ն մինչև Windows XP հարկավոր է իրականացնել հետևյալ քայլերի հաջորդականությունը:

Քայլ 1. Տեղադրեք Windows XP-ի սկավառակը օպտիկական սկավառակատարի մեջ։

Քայլ 2. Ընտրեք Пуск > Выполнить(run)(Menu+R). (սկավառակը օպտիկական սկավառակատարի մեջ դնելուց հետո, տեղադրումը կարող է սկսվել ավտոմատ կերպով)։

Քայլ 3. Եթե օպտիկական սկավառակատարը G-ն է, run-ում հավաքում ենք G:\i386\winnt32 և հաստատում ենք:

Քայլ 4. Ընտրեք «Обновить до Windows XP» և սեղմեք «Далее»(հաջորդը)։ Ցուցադրվում է օգտագործողի լիցենզիոն համաձայնագիրը։

Քայլ 5. Կարդացեք լիցենզիոն համաձայնագիրը և հաստատեք։

Քայլ 6. Սեղմեք «Далее»(հաջորդը)։ Կհայտնվի թարմացնել մինչև Windows XP NTFS ֆայլային համակարգի էջը:
