Графический интерфейс пользователя и панель управления Windows

Администрирование

В окне «Просмотр событий», как показано на рисунке, отображается журнал событий, относящихся к приложениям, безопасности и системе. Файлы журналов являются ценными инструментами поиска и устранения неполадок, поскольку они содержат сведения, необходимые для определения проблемы.

Для получения доступа к функции «Просмотр событий» перейдите по следующим пунктам:

Пуск > Панель управления > Администрирование > Просмотр событий

«Службы компонентов» — это инструмент администрирования, используемый администраторами и разработчиками для развертывания, настройки и администрирования компонентов компонентной объектной модели (COM). Компонентная объектная модель (COM) — это способ обеспечения использования компонентов в средах, отличных от той, в которой они были созданы.

Для получения доступа к функции «Службы компонентов» перейдите по следующим пунктам:

Пуск > Панель управления > Администрирование > Службы компонентов

«Источники данных» — это средство администрирования, используемое для управления источниками данных, их добавления или удаления с помощью интерфейса открытой системы связи с базами данных (ODBC). ODBC — это технология, позволяющая иметь доступ к различным базам и источникам данных.

Для получения доступа к функции «Источники данных (ODBC)» перейдите по следующим пунктам:

Пуск > Панель управления > Администрирование > Источники данных (ODBC)
