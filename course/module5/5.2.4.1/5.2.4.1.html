Установка операционной системы

Многовариантная загрузка

На одном компьютере может находиться несколько операционных систем. Для одних программ может требоваться последняя версия ОС, а для других — более ранняя версия. При наличии на одном компьютере нескольких операционных систем используется процесс двойной загрузки. Во время процесса загрузки диспетчер загрузки Windows (BOOTMGR) определяет наличие более одной ОС и предлагает выбрать ОС для загрузки, как показано на рис. 1.

Для создания системы двойной загрузки в Microsoft Windows жесткий диск должен содержать более одного раздела.

Более раннюю версию ОС следует установить в основной раздел или на жесткий диск, который был выбран в качестве первого активного раздела. Вторую ОС следует установить во второй раздел или на второй жесткий диск. Загрузочные файлы автоматически устанавливаются в активный раздел.

Файл BOOTMGR

Во время установки в активном разделе создается файл BOOTMGR, позволяющий выбрать ОС, загружаемую при запуске. Файл BOOTMGR можно настроить таким образом, чтобы изменить порядок операционных систем. Можно также изменить время выбора ОС на этапе загрузки. Обычно значение по умолчанию составляет 30 секунд. Загрузка компьютера откладывается на указанное время, чтобы дать пользователю возможность выбрать нужную ОС. Если на диске находится только одна ОС, рекомендуется установить время равным 5-10 секундам, чтобы компьютер загружался быстрее.

ПРИМЕЧАНИЕ. В Windows XP вместо файла BOOTMGR используется файл BOOT.INI.

Чтобы изменить время отображения имеющихся операционных систем, перейдите по следующим пунктам:

Нажмите Пуск > Панель управления > Система и безопасность > Система > Дополнительные параметры системы, щелкните вкладку Дополнительно и в поле Загрузка и восстановление выберите пункт Параметры

Чтобы изменить общие данные настройки загрузки в Windows 7 и Vista, используйте средство командной строки bcdedit.exe, как показано на рис. 2. Для доступа к средству bcdedit.exe перейдите по следующим пунктам:

Нажмите Пуск > Все программы > Стандартные, щелкните правой кнопкой мыши пункт Командная строка > Запуск от имени администратора > Продолжить, введите bcdedit.exe

Для изменения файла boot.ini в Windows XP перейдите по следующим пунктам:

Нажмите кнопку Пуск, щелкните правой кнопкой мыши пункт Мой компьютер > Свойства > Дополнительно, в разделе Загрузка и восстановление выберите Параметры и нажмите кнопку Правка
