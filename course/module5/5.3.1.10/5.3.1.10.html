Графический интерфейс пользователя и панель управления Windows

Рабочий стол, инструменты и приложения Windows

В обязанности инженера входит установка ПО на компьютер заказчика и удаление ПО. При установке диска в оптический привод для большинства приложений процесс установки запускается автоматически. В процессе установки выполняется обновление служебной программы «Установка и удаление программ». Во время работы мастера установки пользователь должен по мере необходимости нажимать кнопки и предоставлять запрашиваемую информацию.

Установка приложения

В ОС Windows 7 и Windows Vista вставьте компакт-диск или диск DVD или откройте загруженный файл программы. При этом запустится установщик программы. Если установщик программы не запускается, запустите файл установки с диска, чтобы начать установку, либо повторно загрузите программу.

После установки приложение можно запустить с помощью меню «Пуск» или ярлыка приложения на Рабочем столе. Убедитесь, что приложение работает правильно. В случае возникновения проблем выполните восстановление или удалите приложение с помощью служебной программы «Удаление или изменение программы». В некоторых приложениях, например в Microsoft Office, в процессе установки предусмотрена функция восстановления. Эту функцию можно использовать для попытки исправления неправильно работающей программы.

ПРИМЕЧАНИЕ. В ОС Windows XP, если программа или приложение не устанавливаются автоматически с диска, для их установки можно использовать служебную программу «Установка и удаление программ». Нажмите кнопку Установка новой программы и выберите местоположение приложения.

Удаление и изменение программы

В случае неправильного удаления приложения на жестком диске могут остаться файлы приложения, а в реестре — ненужные параметры, занимающие место на жестком диске и ресурсы системы. Ненужные файлы могут также привести к уменьшению скорости чтения реестра. При удалении, изменении или исправлении приложений Microsoft рекомендует всегда использовать служебную программу «Программы и компоненты», как показано на рисунке. Эта служебная программа управляет процессом удаления ПО и удаляет каждый установленный файл.

В некоторых случаях с помощью служебной программы «Программы и компоненты» можно установить или удалить дополнительные компоненты приложения. Эта возможность есть не у всех программ.

Для получения доступа к служебной программе «Программы и компоненты» в Windows 7 и Windows Vista перейдите по следующим пунктам:

Пуск > Панель управления > Программы и компоненты

В ОС Windows XP перейдите по следующим пунктам:

Пуск > Панель управления > Установка и удаление программ
