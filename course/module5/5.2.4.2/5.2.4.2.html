Установка операционной системы

Многовариантная загрузка

Для настройки многовариантной загрузки требуется несколько жестких дисков или жесткий диск с несколькими разделами. Для создания нового раздела используйте служебную программу управления дисками, как показано на рис.1. Программу управления дисками можно также использовать для выполнения следующих задач:

  • Отображение состояния диска

  • Расширение разделов

  • Разбиение разделов

  • Назначение дискам букв

  • Добавление дисков

  • Добавление массивов

Для получения доступа к служебной программе управления дисками в Windows 7 выполните следующие действия:

нажмите Пуск, щелкните правой кнопкой мыши пункт Компьютер > Управление, выберите пункт Управление дисками

Для получения доступа к служебной программе управления дисками в Windows XP выполните следующие действия:

нажмите Пуск, щелкните правой кнопкой мыши пункт Мой компьютер > Управление, выберите пункт Управление дисками

Состояние диска

Служебная программа управления дисками отображает состояние каждого диска, как показано на рис. 2. Диски компьютера могут находиться в одном из следующих состояний:

  • Чужой — динамический диск, перемещенный в компьютер из другого компьютера, на котором установлена ОС Windows XP

  • Исправен — исправно работающий том

  • Инициализация — идет преобразование базового диска в динамический

  • Отсутствует — динамический диск поврежден, выключен или отключен

  • Не инициализирован — диск не содержит действительной подписи

  • Подключен — базовый или динамический диск доступен и исправен

  • Работает (Ошибки) — на динамическом диске обнаружены ошибки ввода/вывода

  • Не подключен — динамический диск поврежден или недоступен

  • Не читается — базовый или динамический диск, в котором произошел сбой оборудования, повреждение или ошибки ввода/вывода.

При использовании иных дисков, чем жесткие, например звукового компакт-диска в оптическом приводе или пустого съемного диска, отображаются иные индикаторы состояния дисков.

Расширение разделов

С помощью Диспетчера дисков можно расширить основные разделы и логические диски, если на жестком диске имеется незанятое пространство. Для расширения базового диска его необходимо отформатировать в файловой системе NTFS. При расширении жесткого диска увеличивается объем доступного пространства в основном разделе или логическом диске. Логические диски и системные тома должны расширяться в непрерывном пространстве, и тип диска должен быть преобразован в динамический. Другие разделы могут расширяться в несвязанные области, и тип диска должен быть преобразован в динамический

Чтобы расширить раздел с помощью диспетчера дисков, выполните следующие действия:

Этап 1. Щелкните нужный раздел правой кнопкой мыши.

Этап 2. Нажмите кнопку Расширить том.

Этап 3. Следуйте указаниям на экране.

Разбиение разделов

При автоматическом создании разделов на жестком диске в Windows 7 и Windows Vista получается только один раздел. Если нужно разбить раздел, можно использовать функцию сжатия тома. С помощью этой функции размер исходного раздела уменьшается, и образуется свободное место на диске, которое используется для создания нового раздела. Незанятое пространство необходимо отформатировать и назначить ему букву диска.

Для сжатия раздела в Windows 7 и Windows Vista с помощью служебной программы управления дисками щелкните диск правой кнопкой мыши и выберите Сжать том.

Назначение дискам букв

Можно изменить, добавить или удалить букву диска и путь к диску. Windows по умолчанию назначает букву разделу или диску после их создания или добавления. Можно выбрать для названия диска любую букву, если она еще не используется.

Для изменения буквы диска в Windows 7 с помощью служебной программы управления дисками щелкните диск правой кнопкой мыши и выберите Изменить букву диска или путь к диску.

Добавление дисков

Для увеличения объема пространства для хранения данных на компьютере или для реализации настройки RAID к компьютеру можно добавить диски. Если дополнительный жесткий диск установлен правильно, BIOS должен автоматически распознать его. После установки диска проверьте с помощью служебной программы управления дисками, распознан ли он. Если диск доступен, возможно, перед его использованием требуется форматирование. Если диск не отображается, устраните эту проблему.

Добавление массивов

Для настройки RAID необходимо установить на компьютер два или более дисков. Добавлять массивы можно с помощью служебной программы управления дисками. Существуют следующие варианты добавления массивов:

  • Новый составной том — создание раздела диска, состоящего из дискового пространства нескольких физических дисков. Данные на составном томе не являются отказоустойчивыми.

  • Новый чередующийся том — динамический раздел, на котором данные хранятся в виде полос на нескольких физических дисках. Данные на чередующемся томе не являются отказоустойчивыми.

  • Новый зеркальный том — дублирует данные с одного диска на один или несколько дополнительных физических дисков. Данные на зеркальном томе отказоустойчивы.

  • Новый том RAID-5 — динамический раздел, данные на котором хранятся в виде полос на нескольких физических дисках и обеспечивает проверку четности для каждой полосы. Данные тома RAID-5 отказоустойчивы.

ПРИМЕЧАНИЕ. Возможности добавления массивов ограничены системой. Не все возможности могут быть доступны.

Для добавления массива с помощью служебной программы управления дисками щелкните нужный диск правой кнопкой мыши и выберите нужный вариант.
