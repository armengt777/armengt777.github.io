Служебные программы панели управления

Для обеспечения надлежащего доступа к файлам требуется управление параметрами каталогов и папок. Для настройки параметров папок в Windows используется служебная программа «Параметры папок».

Для получения доступа к служебной программе «Параметры папок» в Windows 7 используйте следующие пункты меню:

Пуск > Панель управления > Параметры папок

На рисунке показано, что в программе «Параметры папок» есть три вкладки.

Общие

Вкладка «Общие» используется для настройки базовых параметров отображения и доступа.

  • Обзор папок — настройка отображения папки при ее открытии.

  • Щелчки мышью — количество щелчков мыши для открытия файла.

  • Область переходов — указывает, все ли папки отображаются и происходит ли автоматическое развертывание папки при ее выборе в области переходов.

Вид

Вкладка «Вид» используется для настройки параметров и атрибутов просмотра папок, включая возможность просмотра скрытых папок.

  • Представление папок — применение параметров просмотра последней папки ко всем однотипным папкам.

  • Дополнительные параметры — настройка дополнительных параметров просмотра.

Поиск

Вкладка «Поиск» используется для настройки поиска папок.

  • Что искать — настройка поиска на основе индексированных или неиндексированных местоположений для облегчения поиска файлов и папок.

  • Как искать — определение критериев, принимаемых во внимание при поиске.

  • При поиске в неиндексированных расположениях — указывается, включаются ли системные папки и сжатые файлы при поиске в неиндексированных расположениях.
