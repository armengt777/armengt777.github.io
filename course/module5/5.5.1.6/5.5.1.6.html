Стандартные методы профилактического обслуживания операционных систем

План профилактического обслуживания операционной системы

Можно запланировать время запуска программ профилактического обслуживания. Планировать задачи можно с помощью планировщика заданий Windows на основе графического интерфейса пользователя или с помощью команды интерфейса командной строки at. Выбрав любой из этих способов, можно однократно запустить команду в указанное время или многократно запускать ее в установленные дни и время. Планировщик заданий Windows, изображенный на рисунке, проще использовать для выполнения повторяющихся задач и удаления запланированных задач, чем команду at.

Планировщик заданий Windows

С помощью планировщика заданий можно автоматизировать выполнение задач. Планировщик заданий отслеживает определенные пользователем критерии и выполняет задачи, которые соответствуют выбранным критериям. Вот некоторые задачи, которые выполняются автоматически с помощью планировщика заданий:

  • очистка диска;

  • создание резервных копий;

  • дефрагментация диска;

  • создание точки восстановления;

  • запуск других приложений.

Для получения доступа к планировщику заданий в ОС Windows 7 и Windows Vista перейдите по следующим пунктам:

Пуск > Все программы > Стандартные > Служебные программы > Планировщик заданий

В ОС Windows XP перейдите по следующим пунктам:

Пуск > Все программы > Стандартные > Служебные > Назначенные задания

Команда at

Команда at используется для планового выполнения команд, файлов сценариев и приложений. Чтобы использовать команду at, необходимо войти в систему в качестве администратора.

Для получения дополнительной информации о команде at в ОС Windows 7 и Windows Vista выполните следующие действия:

Нажмите Пуск > окно поиска, введите cmd и нажмите клавишу Ввод, введите at /? в командной строке и нажмите клавишу Ввод

Для получения информации о команде at в ОС Windows XP выполните следующие действия:

Нажмите Пуск > Выполнить, введите cmd и нажмите клавишу Ввод, введите at /? в командной строке и нажмите клавишу Ввод
