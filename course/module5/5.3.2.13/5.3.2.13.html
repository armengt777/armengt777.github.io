Графический интерфейс пользователя и панель управления Windows

Служебные программы панели управления

Помимо настроек безопасности Центра поддержки, атаки на компьютер можно предотвратить с помощью служебной программы «Брандмауэр Windows», показанной на рисунке. Брандмауэр применяет политику безопасности, выборочно разрешающую или блокирующую передачу данных на компьютер. Название «брандмауэр» происходит от названия противопожарной стены из кирпича и цемента, предназначенной для предотвращения распространения огня из одной части здания в другую.

Параметры брандмауэра можно настроить для домашних, корпоративных и общественных сетей. Для дополнительных изменений используются следующие пункты:

  • Разрешить запуск программы или компонента через брандмауэр Windows — определяет программы, взаимодействующие с брандмауэром Windows.

  • Изменение параметров уведомления — пользователи с правами администратора могут настраивать уведомления брандмауэра Windows.

  • Включение или отключение брандмауэра Windows — пользователи с правами администратора могут включать или отключать брандмауэр Windows.

  • Восстановить умолчания — пользователи с правами администратора могут восстановить параметры по умолчанию брандмауэра Windows.

  • Дополнительные параметры — пользователи с правами администратора могут устанавливать дополнительные параметры безопасности.

Для получения доступа к служебной программе «Брандмауэр Windows» в ОС Windows 7 перейдите по следующим пунктам:

Пуск > Панель управления > Брандмауэр Windows
