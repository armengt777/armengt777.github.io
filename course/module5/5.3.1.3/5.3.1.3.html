Графический интерфейс пользователя и панель управления Windows

Рабочий стол, инструменты и приложения Windows

Меню «Пуск» и панель задач позволяют пользователям управлять программами, выполнять поиск в компьютере и работать с запущенными приложениями. Для настройки меню «Пуск» или панели задач щелкните компонент правой кнопкой мыши и выберите пункт Свойства.

Меню «Пуск»

Доступ в меню «Пуск» осуществляется с рабочего стола нажатием значка Windows в нижнем левом углу рабочего стола. В меню «Пуск», как показано на рис. 1, отображаются все установленные на компьютере приложения, список недавно открытых документов и список других элементов, например функции поиска, справки и поддержки и параметров системы. Можно настроить различные компоненты меню «Пуск», как показано на рис. 2. Для настройки параметров меню «Пуск» выполните следующие действия:

Щелкните правой кнопкой мыши пустую область панели задач и перейдите по пунктам Свойства > Меню «Пуск» > Настроить

В Windows XP можно выбрать один из стилей меню «Пуск»: XP или Классический. В Windows 7 вариант выбора стилей меню «Пуск» удален.

Панель задач

В Windows 7 к панели задач добавлены новые функции, упрощающие переходы между окнами, упорядочение окон и доступ к окнам и уведомлениям:

  • Список переходов — для отображения списка задач, свойственных только конкретному приложению, щелкните значок приложения в панели задач правой кнопкой мыши.

  • Закрепленные приложения — упрощает доступ к приложению, добавляя его в панель задач. Для этого щелкните значок приложения правой кнопкой мыши и выберите пункт Закрепить на панели задач.

  • Просмотр эскизов — для просмотра эскиза запущенной программы наведите курсор на значок программы на панели задач.
