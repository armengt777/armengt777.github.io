Ժամանակակից օպերացիոն համակարգեր

ՕՀ-ի տեսակները

Գոյություն ունեն երկու տեսակի ՕՀ-եր՝ սեղանի(Desktop) և ցանցային: Desktop ՕՀ-երը նախատեսված են տան և փոքր գրասենյակների համար (small office, home office - SOHO) , որոնք ունեն սահմանափակ քանակությամբ օգտագործողներ: Ցանցային ՕՀ-երը (NOS- Network Ooperating System) նախատեսված եմ կորպորատիվ միջավայրի համար, որտեղ սպասարկում են բազմաթիվ օգտագործողների լայն կարիքները:

Desktop ՕՀ-ն ունի հետևյալ բնութագրերը.

  • Աջակցում է միայն մեկ օգտագործողի;

  • Աշխատում է մեկ օգտագործողի աշխատածրագրերի(կիրառական ծրագրերի) հետ;

  • Կիսում է փոքր ցանցերի ֆայլերը և թղթապանակները սահմանապակ անվտանգությամբ (անվտանգության ապահովմամբ)

Հաճախակի օգտագործվող Desktop ՕՀ-ները կարելի է բաժանել 3 խմբի՝ Microsoft Windows, Apple Mac OS և Linux:

Microsoft Windows

Windows-ը ներկայումս համարվում է ամենատարածված ՕՀ-ներից մեկը: Ստորև ներկայացված են windows-ի հետևյալ տեսակները․

  • Windows 7 Начальная(Starter) — օգտագործվում է նոութբուքների համար, հեշտացնում է ցանցային աշխատանքը:( Used on netbook computers to make networking easy)

  • Windows 7 Домашняя расширенная(Home Premium) — օգտագործվում է տնային համակարգիչների համար, հեշտությամբ կիսում է միջավայրը:( Used on home computers to easily share media)

  • Windows 7 Профессиональная(Professional) — օգտաործվում է փոքր բիզնեսի համակարգիչների համար, ապահովում է կարևոր ինֆորմացիան և դարցնում է ընթացիկ խնդիրները ավելի հեշտ ավարտելի:(Used on small business computers to secure critical information and to make routine tasks easier to complete)

  • Windows 7 Корпоративная(Enterprise) — օգտագործվում է մեծ բիզնեսի համակարգիչների համար, ապահովում է բարձր արտադրողականություն, անվտանգություն և ղեկավարման ֆունկցիաներ:

  • Windows 7 Максимальная(Ultimate) — օգտագործվում է համակարգիչների վրա, ընդգրկում է իր մեջ Windows 7 Home Premium-ը Windows 7 Professional-ի բիզնես հնարավորություններով և ապահովում է տվյալների լրացուցիչ անվտանգություն

  • Windows Vista Home Basic — օգտագործվում է տնային համակարգիչների կողմից, հիմնական հաշվարկների համար

  • Windows Vista Home Premium — օգտագործվում է տնային համակարգիչների համար, ընդլայնում է անհատական արտադրողականությունը և հիմնական աշխատանքից բացի թվային ժամանցի հնարավորությունը

  • Windows Vista Business — օգտաործվում է փոքր բիզնեսի համակարգիչների համար, անվտանգության բարձրացման և շարժունակության տեխնոլոգիայի ընդլայնման համար

  • Windows Vista Enterprise — օգտագործվում է մեծ բիզնեսի համակարգիչների համար, ապահովում է բարձր արտադրողականություն, անվտանգություն և ղեկավարման ֆունկցիաներ

  • Windows Vista Ultimate — օգտագործվում է համակարգիչների վրա, որպեսզի միավորի տնային և բիզնես օգտագործողների պահանջները

  • Windows XP Professional — մեծամասամբ օգտագործվում է այն համակարգիչների համար, որոնք միացված են Windows Server-ին ցանցում

  • Windows XP Home — օգտագործվում է տնային համակարգիչների համար և ունի սահմանափակ անվտանգություն

  • Windows XP Media Center — иօգտագործվում է ժամանցային նպատակներով՝ ֆիլմեր նայելու, երաժշտություն լսելու համար

  • Windows XP 64-bit Professional — иօգտագործվում է 64 բիթանոց պրոցեսրով համակարգիչների համար

Apple Mac OS

Apple համակարգիչները համարվում են սեփականություն և օգտագործում են Mac OS օպերացիոն համակարգը: Mac OS -ը ունի պարզ գրաֆիկական ինտերֆեյս: Mac OS-ի ընթակիկ տարբերակները հիմնված են UNIX-ի հատուկ տարբերակի վրա:

Linux

Linux ստեղծվել է UNIX-ի հիման վրա, որն էլ ստեղծվել է 1960-ականների վերջին և համարվում է հնագույն ՕՀ-ներից մեկը: Լինուս Տորվալդը 1991թ.-ին մշակեց Linux-ը, որպես բաց կոդով ՕՀ: Բաց ընթացիկ կոդով ծրագրերը թույլ են տալիս ,որ բաց կոդը տարածվի և փոփոխվի ցանկացածի կողմից:
