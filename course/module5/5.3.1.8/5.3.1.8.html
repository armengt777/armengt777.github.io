Графический интерфейс пользователя и панель управления Windows

Рабочий стол, инструменты и приложения Windows

Компьютер

Пункт главного меню «Компьютер» обеспечивает доступ к жестким дискам и приводам, установленным на компьютере. В Windows 7 и Windows Vista нажмите Пуск и выберите пункт Компьютер. В Windows XP этот пункт называется «Мой компьютер». Доступ к нему можно получить, дважды щелкнув значок Мой компьютер на Рабочем столе.

Проводник Windows

Проводник Windows обеспечивает навигацию по файловой системе. На рисунке изображена область переходов для Windows 7. Вверху отображается путь к доступным папкам и файлам, вверху справа — панель поиска, панель инструментов расположена непосредственно под панелью пути и поиска, область переходов — слева, а панель просмотра — справа от области перехода. Панель поиска используется для поиска конкретного приложения, файла или папки. Панель инструментов позволяет упорядочивать файлы, добавлять новые папки, изменять расположение файлов в Проводнике Windows, отображать панель просмотра для файлов и папок и открывать справку Windows. Область переходов содержит список папок по умолчанию, в числе которых «Избранное», «Библиотеки», «Компьютер» и «Сеть». Панель просмотра используется для доступа к приложениям, файлам и папкам и управления ими.

В Windows 7 Проводник Windows поддерживает следующие новые функции:

  • Окно поиска — доступ к прежним результатам поиска. Результаты поиска можно также фильтровать.

  • Кнопка «Новая папка» — создание новых папок одним щелчком мыши.

  • Упорядочить — упорядочение элементов по различным критериям.

  • Кнопка «Область просмотра» — включение или отключение области просмотра при необходимости.

Запуск приложений и открытие файлов

Существует несколько способов запуска приложений:

  • Щелкните значок приложения в меню «Пуск» или на панели задач.

  • Дважды щелкните запускаемое приложение или его ярлык на Рабочем столе или в Проводнике Windows.

  • Запустите приложение с помощью окна запуска или командной строки.

Этот же способ можно использовать для открытия файлов. При открытии файла Windows определяет приложение, связанное с этим файлом. Windows сопоставляет расширение файла с установленными на компьютере приложениями, с помощью которых можно открыть этот файл. Например, при открытии файла .docx Windows запускает приложение Microsoft Word.
