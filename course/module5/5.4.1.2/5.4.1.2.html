Назначение и требования виртуализации

Программа для создания виртуальной машины и управления ею на главном компьютере называется гипервизором, или диспетчером виртуальных машин (Virtual Machine Manager, VMM). Гипервизор может запускать несколько виртуальных машин на одном главном компьютере. Каждая виртуальная машина работает под управлением собственной операционной системы. Количество доступных виртуальных машин зависит от аппаратных ресурсов главного компьютера. Гипервизор назначает необходимые физические системные ресурсы (ЦП, ОЗУ и жесткий диск) для каждой виртуальной машины. Это предотвращает возникновение конфликтов при работе различных виртуальных машин.

Существует два типа гипервизоров: тип 1 (автономный) и тип 2 (на основе базовой ОС). Гипервизор типа 1 запускается непосредственно на оборудовании главного компьютера и управляет системными ресурсами гостевых операционных систем. Гипервизор типа 2 запускается в ОС главного компьютера. Windows Virtual PC является примером гипервизора типа 2.

Windows Virtual PC

Windows Virtual PC представляет собой платформу виртуализации для ОС Windows 7. Он позволяет распределять системные ресурсы, предназначенные для ОС Windows, среди виртуальных машин, работающих с лицензированными копиями ОС Windows 7, Windows Vista или Windows XP. Virtual PC можно загрузить с веб-сайта Microsoft Windows. На рисунке раздела 5.4.1.3 приведены требования к оборудованию для запуска Virtual PC. Virtual PC требуется для запуска программы «Режим Windows XP» в Windows 7.

Режим Windows XP

Режим Windows XP — это программа, доступная для версий ОС Windows 7 Профессиональная, Корпоративная и Максимальная. В режиме Windows XP используется технология виртуализации, позволяющая пользователям запускать программы Windows XP из Windows 7. С его помощью на Рабочем столе Windows 7 открывается виртуальная машина, предоставляющая полнофункциональную версию ОС Windows XP, включая доступ ко всем системным ресурсам. После установки программы в режиме Windows XP можно запускать эту программу в режиме XP и получать к ней доступ из меню «Пуск» ОС Windows 7.

ПРИМЕЧАНИЕ. Прежде чем использовать режим Windows XP, загрузите и установите Windows Virtual PC.

Для получения доступа к режиму XP в Windows 7 выполните следующие действия:

Этап 1. Выберите пункты Пуск > Все программы.

Этап 2. Выберите Windows Virtual PC > Режим Windows XP.
