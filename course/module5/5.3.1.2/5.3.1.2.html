Рабочий стол, инструменты и приложения Windows

Для настройки рабочего стола в Windows 7 и Windows Vista щелкните правой кнопкой мыши любую точку рабочего стола и выберите пункт Персонализация. В Windows 7 в окне персонализации, как показано на рис. 1, слева находятся три ссылки, позволяющие выбрать значки рабочего стола, указатели мыши и изображение для учетной записи. Темы выбираются из правой панели. Можно изменить фон, цвет, звук темы, а также экранную заставку с помощью четырех элементов, находящихся в нижней части этого окна.

В ОС Windows Vista окно персонализации содержит семь ссылок, как показано на рис. 2. Они позволяют настраивать цвет и вид окна, изменять фон рабочего стола, устанавливать параметры экрана и многое другое. Каждая ссылка служит для персонализации различных компонентов рабочего стола.

Для настройки рабочего стола в Windows XP щелкните правой кнопкой мыши любую точку рабочего стола и выберите пункт Свойства. В окне «Свойства экрана» отображаются различные вкладки, как показано на рис. 3. Каждая вкладка служит для персонализации различных параметров отображения.
