Программы командной строки

Запуск служебных команд из командной строки (см. рисунок 1) позволяет вводить команды для настройки параметров в ОС Windows. Многие из этих команд используются для диагностики системы и внесения изменений.

Для получения доступа к запуску программ из командной строки в Windows 7 выполните следующие действия:

нажмите Пуск > окно поиска, введите run > Ввод

Для получения доступа к запуску команд из командной строки в Windows Vista выполните следующие действия:

нажмите Пуск > Начать поиск, введите run > Ввод

Для получения доступа к запуску команд из командной строки в Windows XP выполните следующие действия:

нажмите Пуск > командная строка

Ниже приведен список стандартных команд.

  • CMD — используется для выполнения программ и служебных программ командной строки (см. рисунок 2).

  • DXDIAG — отображает сведения обо всех драйверах и компонентах DirectX, установленных на компьютере (см. рисунок 3). Использование этой команды обеспечивает правильную установку и настройку DirectX.

  • EXPLORER — открывает проводник Windows.

  • MMC — открывает консоль управления (MMC) (см. рисунок 4), позволяющую упорядочить инструменты управления (интегрированные приложения) в одном местоположении для упрощения администрирования. Можно также добавлять ссылки на веб-страницы, задачи, элементы управления ActiveX и папки. Можно создавать необходимое количество консолей управления с различными настройками под разными именами. Эта функция полезна в том случае, управлением компьютером выполняется различными администраторами. У каждого администратора есть индивидуальная консоль управления для отслеживания и настройки параметров компьютера. Для получения доступа к консоли управления необходимо обладать правами администратора.

  • MSCONFIG — открывает служебную программу «Настройка системы» (см. рисунок 5), выполняющую процедуры диагностики файлов запуска Windows. Для выполнения процедур устранения неполадок необходимо войти в систему с правами администратора. Команда MSCONFIG используется в том случае, если компьютер загружается, но загрузка Windows происходит некорректно.

  • MSINFO32 — отображает сведения о системе, включая компоненты оборудования и информацию о ПО (см. рисунок 6).

  • MSTSC — открывает подключение к удаленному рабочему столу.

  • NOTEPAD — открывает программу «Блокнот» (см. рисунок 7), которая является базовым текстовым редактором.

  • REGEDIT — открывает служебную программу «Редактор реестра» (см. рисунок 8), позволяющую пользователю вносить изменения в реестр. Неправильное использование редактора реестра может привести к неисправностям оборудования, приложений или операционной системы, включая проблемы, для устранения которых требуется переустановка операционной системы.

  • SERVICES.MSC — открывает служебную программу «Службы».

MSCONFIG

Команда MSCONFIG называется «Конфигурация системы» в Windows 7 и Windows Vista, а в Windows XP она называется «Настройка системы». Настройка системы может использоваться для отключения или повторного подключения драйверов устройств, служб Windows и программ. С ее помощью можно также изменить параметры загрузки.

Для управления различными функциями используются следующие пять вкладок:

  • Общие— используется для выбора способов запуска: обычный, диагностический и выборочный.

  • Загрузка — отображает варианты загрузки, которые можно включить или отключить.

  • Службы — используется для отключения и включения текущих служб

  • Автозагрузка — используется для отключения и включения элементов, которые автоматически загружаются при включении компьютера

  • Сервисы — используется для запуска функций Windows, которые обычно находятся на панели управления
