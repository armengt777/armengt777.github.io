Служебные программы

Для обслуживания и оптимизации операционной системы Windows доступны различные средства. Среди них — дефрагментация жесткого диска, благодаря которой файлы подвергаются переразмещению для более быстрого доступа к ним, а также проверка диска, позволяющая обнаружить ошибки в структуре файлов на жестком диске.

ОС Windows содержит служебные программы для поддержки целостности системы. Среди них — две служебные программы, используемые для профилактического обслуживания: дефрагментация диска и средство проверки диска (CHKDSK).

Дефрагментация диска

По мере увеличения размера файла некоторые данные записываются в ближайший доступный кластер на диске. Со временем данные становятся фрагментированными и занимают несмежные кластеры жесткого диска. В результате поиск отдельных разделов данных занимает больше времени. При дефрагментации диска выполняется сбор несмежных данных в одно местоположение, что ускоряет работу операционной системы. В ОС Windows 7 программа дефрагментации диска (см. рисунок 1) настроена на автоматический запуск в среду утром или при следующем включении компьютера.

ПРИМЕЧАНИЕ. Не рекомендуется выполнять дефрагментацию диска Windows для твердотельных дисков. Для оптимизации твердотельных дисков используется контроллер и специальные микропрограммы. Определить, является ли жесткий диск твердотельным, можно при просмотре доступных дисков в диспетчере устройств, как показано на рисунке 2.

Для получения доступа к программе дефрагментации диска в Windows 7 перейдите по следующим пунктам:

Пуск > Все программы > Стандартные > Служебные > Дефрагментация диска

В ОС Windows Vista перейдите по следующим пунктам:

Пуск > Компьютер, щелкните правой кнопкой мыши Диск x > Свойства > Инструменты

В ОС Windows XP перейдите по следующим пунктам:

Пуск > Все программы > Стандартные > Служебные > Дефрагментация диска

Средство проверки диска

Средство проверки диска используется для проверки целостности файлов и папок путем сканирования поверхности жесткого диска на физические ошибки. При обнаружении ошибок эта программа устраняет их. Доступ к программе проверки диска можно получить через программу дефрагментации диска или введя "CHKDSK" в командную строку. Проверку диска на ошибки можно также выполнить следующим образом:

Этап 1. Нажмите кнопку Пуск, выберите пункт Компьютер.

Этап 2. Щелкните выбранный диск правой кнопкой мыши и нажмите Свойства.

Этап 3. Выберите вкладку Сервис .

Этап 4. В пункте «Проверка диска» выберите Выполнить проверку.

Этап 5. В параметрах проверки диска выберите Проверять и восстанавливать поврежденные сектора.

Средство проверки диска исправляет ошибки файловой системы и проверяет диск на наличие поврежденных секторов. Оно также пытается восстановить данные поврежденных секторов.

ПРИМЕЧАНИЕ. Рекомендуется выполнять проверку диска не менее одного раза в месяц, а также при внезапной остановке системы из-за отключения электричества.
