Графический интерфейс пользователя и панель управления Windows

Программы панели управления для отдельных версий Windows

Некоторые функции панели управления доступны только в версии Windows Vista, среди них:

  • Параметры планшетного ПК

  • Перо и устройства ввода

  • Автономные файлы

  • Отчеты о проблемах и их решениях

  • Принтеры

Параметры планшетного ПК

Служебная программа «Параметры планшетного ПК» (см. рисунок 1) позволяет настраивать функции планшетного ПК. Для получения доступа к программе «Параметры планшетного ПК» перейдите по следующим пунктам:

Пуск > Панель управления > Параметры планшетного ПК

Перо и устройства ввода

Программа «Перо и устройства ввода» (см. рисунок 2) позволяет выполнять настройки цифрового пера и других устройств ввода. Для получения доступа к служебной программе «Перо и устройства ввода» перейдите по следующим пунктам:

Пуск > Панель управления > Перо и устройства ввода

Автономные файлы

Служебная программа «Автономные файлы» (см. рисунок 3) позволяет выбирать совместно используемые в сети файлы и папки и сохранять их на своем компьютере. Эти файлы остаются доступными после отключения компьютера от сети. При повторном подключении к сети изменения, внесенные в автономном режиме, применяются к исходным файлам в сети.

Для настройки компьютера на использование автономных файлов и папок перейдите по следующим пунктам:

Пуск > Панель управления > Автономные файлы, щелкните вкладку Общие, выберите пункт Включить автономные файлы

Для просмотра списка всех автономных файлов выполните следующие действия:

Пуск > Панель управления > Автономные файлы, щелкните вкладку Общие, выберите пункт Просмотреть автономные файлы

Отчеты о проблемах и их решениях

Служебная программа «Отчеты о проблемах и их решениях» (см. рисунок 4) ведет журнал проблем, которые возникали в системе, и решений, предложенных Microsoft. В Windows 7 эта функция обеспечивается центром поддержки.

Для получения доступа к служебной программе «Отчеты о проблемах и их решениях» перейдите по следующим пунктам:

Пуск > Панель управления > Отчеты о проблемах и их решениях

Принтеры

Служебная программа «Принтеры» (см. рисунок 5) позволяет добавлять, удалять и настраивать принтеры. Для получения доступа к служебной программе «Принтеры» перейдите по следующим пунктам:

Пуск > Панель управления > Принтеры
