Графический интерфейс пользователя и панель управления Windows

Программы панели управления для отдельных версий Windows

Некоторые функции панели управления доступны только в версии Windows 7, среди них:

  • Домашняя группа

  • Центр поддержки

  • Защитник Windows

  • Подключения к удаленным рабочим столам и приложениям RemoteApp

  • Устранение неполадок

Домашняя группа

Домашняя группа — это сетевая настройка для упрощения пользователям общего доступа к файлам и папкам в домашней сети. Служебная программа «Домашняя группа» (см. рисунок 1) недоступна в рабочей или публичной сети.

Центр поддержки

Служебная программа «Центр поддержки» используется вместо программы «Центр обеспечения безопасности», доступной в версиях ОС XP и Vista. Подробное описание программы «Центр поддержки» можно найти в разделе «Центр поддержки», представленном в данном курсе.

Защитник Windows

Защитник Windows — это антишпионская программа, проверяющая операционную систему на наличие шпионского ПО, которое может представлять угрозу безопасности. Она защищает компьютер от вредоносного ПО, например шпионских программ, созданных для атак злоумышленников. В случае обнаружения Защитником Windows нежелательного ПО пользователь может удалить вредоносные файлы или поместить их на карантин. Для получения доступа к Защитнику Windows используйте следующие пункты меню:

Пуск > Панель управления > Защитник Windows

Защитник Windows по умолчанию доступен в ОС Windows 7 и Windows Vista и может быть загружен в Windows XP.

Подключения к удаленным рабочим столам и приложениям RemoteApp

Служебная программа «Подключения к удаленным рабочим столам и приложениям RemoteApp» (см. рисунок 2) обеспечивает доступ к удаленным компьютерам и программам с одной папки в проводнике Windows. Для получения доступа к служебной программе «Подключения к удаленным рабочим столам и приложениям RemoteApp» перейдите по следующим пунктам:

Пуск > Панель управления > Подключения к удаленным рабочим столам и приложениям RemoteApp

Устранение неполадок

Служебная программа «Устранение неполадок» (см. рисунок 3) позволяет устранять проблемы в следующих категориях:

  • Программы — устранение проблем совместимости с программами, созданными для предыдущих версий Windows.

  • Оборудование и звук — диагностика и устранение проблем настройки устройств и устранение неполадок воспроизведения звука

  • Сеть и Интернет — устранение проблем подключения к Интернету и доступа к общим файлам и папкам.

  • Оформление и персонализация — устранение проблем, связанных с оформлением Рабочего стола.

  • Система и безопасность — выполнение задач обслуживания, поиск проблем производительности и оптимизация энергопотребления.

Для получения доступа к служебной программе «Устранение неполадок» перейдите по следующим пунктам:

Пуск > Панель управления > Устранение неполадок
