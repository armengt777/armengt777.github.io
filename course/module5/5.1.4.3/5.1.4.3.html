Ժամանակակից օպերացիոն համակարգեր

ՕՀ-ների թարմացում

Երբ պահանջվում է ՕՀ-ի նոր տեղադրում, հարկավոր է օգտագործողի տվյալները հին ՕՀ-ից տեղափոխել նոր ՕՀ: Գոյություն ունեն երեք հասանելի գործիքներ, որոնց միջոցով իրականացվում է տվյալների և կարգավորումների տեղափոխման գործընթացը: Գործիքի ընտրությունը կախված է ձեր փորցից և պահանջներից:

Средство миграции пользовательской среды (User State Migration Tool(USMT)

USMT-ն տեղափոխում է օգտագործողի ողջ տվյալները և կարգավորումները նոր ՕՀ, ինչպես ցույց է տրված նկար 1․-ում. Բեռնավորեք և տեղադրեք USMT-ն microsoft-ի կայքից: Այնուհետև օգտագործելով ծրագրային ապահովումը ստեղծեք օգտագործողի տվյալների և կարգավորումների պահոց (store), որոնք պահվում են ՕՀ-ից տարբեր վայրում: Նոր Օհ-ի տեղադրումից հետո կրկին բեռնավորեք և տեղադրեք USMT-ն օգտագործողի տվյալները և կարգավորումները նոր Օհ-ի վրա բեռնավորելու համար:

Средство переноса данных Windows (Windows Easy Transfer)

Օգտագործողի ողջ տվյալները և կարգավորումները նոր ՕՀ տեղափոխելու համար կարող եք օգտագործոլ նաև Windows Easy Transfer գործիքը նկ․-2։ Դուք կարող եք կատարել ֆայլերի տեղափոխում օգտագործոլով USB մալուխ, CD կամ DVD, USB Flash կրիչ, արտաքին կրիչ կամ ցանցային միացումը:

Windows Easy Transfer-ի աշխատանքից հետո կարող եք տեսնել տեղափոխված ֆայլերի մասին գրանցումները: Windows 7 կամ Windows Vista ՕՀ-երում Windows Easy Transfer-ին դիմելու համար օգտագործեք հետևյալ ճանապարհը՝

Пуск > Все программы > Стандартные > Служебные > Средство переноса данных Windows

Start > All Programs > Accessories > System Tools > Windows Easy Transfer

Windows XP ՕՀ-ի դեպքում հարկավոր է նախ բեռնել Windows Easy Transfer ծրագիրը: Բեռնելուց հետո օգտվելու համար օգտագործեք հետևյալ ճանապարհը՝

Пуск > Все программы > Средство переноса данных Windows

Start > All Programs > Windows Easy Transfer

Տվյալների և կարգավորումների տեղափոխման ծրագիր Windows XP -ի համար

Мастер переноса файлов и параметров для Windows XP

(Files and Settings Transfer Wizard)

Windows XP-ի համար տվյալների և կարգավորումների տեղափոխման ծրագիրը պատկերված է նկ․-3։ Այն թույլ է տալիս ընտրել այն ֆայլերը և կարգավորումները, որոնք պետք է տեղափոխվեն:Դուք կարող եք ֆայլերը տեղափոխել օգտագործելով կրիչներ, մալուխային միացումը կամ ցանցային միացումը: տվյալների և կարգավորումների տեղափոխման ծրագրից օգտվելու համար օգտագործեք հետևյալ ճանապարհը

Пуск > Все программы > Стандартные > Служебные > Мастер переноса файлов и параметров

Start > All Programs > Accessories > System Tools > Files and Settings Transfer Wizard
