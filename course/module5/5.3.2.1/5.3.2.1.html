Графический интерфейс пользователя и панель управления Windows

Служебные программы панели управления

ОС Windows обеспечивает централизованный доступ к параметрам многих функций, которые управляют работой и содержимым компьютера. Эти параметры объединены в служебные, или малые программы, которые расположены на панели управления, как показано на рисунке 1. Установка или удаление программ, изменение параметров сети и безопасности представляют собой примеры функций настройки, доступных на панели управления.

Имена категорий панели управления незначительно различаются в различных версиях Windows. В Windows 7 значки по умолчанию объединены в восемь следующих категорий:

  • Система и безопасность — настройка параметров системы и безопасности

  • Сеть и Интернет — настройка типов сетевого подключения

  • Оборудование и звук — настройка подключенных к компьютеру устройств и параметров звука

  • Программы — удаление, изменение и исправление ошибок в приложениях

  • Учетные записи пользователей и семейная безопасность — создание и удаление учетных записей пользователя и настройка родительского контроля

  • Оформление и персонализация — управление настройками графического интерфейса пользователя Windows

  • Часы, язык и регион — установка местоположения и языка

  • Специальные возможности — настройка Windows при наличии у пользователя ограничений зрения, слуха или подвижности

Windows позволяет изменить отображение панели управления. Выбранный тип представления определяет программы, которые будут доступны непосредственно с панели управления. В Windows 7 доступны следующие варианты представления:

  • Категория — служебные программы панели управления объединены в группы для облегчения переходов.

  • Крупные значки — служебные программы отображены в алфавитном порядке с помощью крупных значков.

  • Мелкие значки — служебные программы отображены в алфавитном порядке с помощью мелких значков.

ПРИМЕЧАНИЕ. В данном курсе используется представление с крупными значками (см. рисунок 2).

В Windows Vista доступны два варианта представления:

  • Панель управления — домашняя страница — служебные программы панели управления объединены в группы для облегчения переходов.

  • Классический вид — все служебные программы панели управления отображаются отдельно.

В Windows XP доступны два варианта представления:

  • Вид по категориям — служебные программы панели управления объединены в группы, удобные для переходов.

  • Классический вид — все служебные программы панели управления отображаются отдельно.
