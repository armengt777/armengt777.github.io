Специализированные компьютерные системы

Для сборки домашнего кинотеатра на основе ПК (Home Theater Personal Computer — HTPC) требуется специализированное оборудование, обеспечивающее высококачественное изображение для пользователя. Каждый компонент оборудования должен подключаться и надлежащим образом предоставлять необходимые сервисы и ресурсы для поддержки различных требований к системе HTPC.

Полезной функцией HTPC является возможность записи видеопрограмм для просмотра в будущем. Системы HTPC могут быть разработаны для просмотра телевизионных программ в прямом эфире, потоковой передачи фильмов и содержимого Интернета, просмотра семейных фотографий и видеозаписей и даже для работы в Интернете c использованием телевизора в качестве монитора. При сборке HTPC может понадобиться следующее оборудование.

  • Специализированные корпуса и источники питания — при сборке HTPC можно использовать материнские платы меньшего размера, чтобы компоненты могли поместиться в корпусе более компактного форм-фактора. Этот корпус с маленьким форм-фактором выглядит как типичный компонент домашнего кинотеатра. Как правило, в корпусе HTPC устанавливаются большие вентиляторы, которые вращаются медленнее и создают меньше шума по сравнению с вентиляторами в типичной рабочей станции. Чтобы дополнительно снизить шум, создаваемый системой HTPC, можно использовать источники питания без вентиляторов (в зависимости от требований к питанию). Некоторые конструкции HTPC содержат компоненты с особо низким энергопотреблением и не требуют вентиляторов для охлаждения.

  • Аудиосистема объемного звука — объемный звук создает эффект присутствия в видеопрограмме. В системе HTPC может использоваться функция объемного звука материнской платы (если ее поддерживает чипсет), либо можно установить специализированную звуковую плату для передачи высококачественного объемного звука на динамики или дополнительный усилитель для оптимального звучания.

  • Выходной разъем HDMI — стандарт HDMI позволяет передавать видео в формате высокой четкости, объемный звук и данные на телевизоры, мультимедийные ресиверы и проекторы.

  • ТВ-тюнеры и кабельные платы — для того, чтобы система HTPC отображала телевизионные сигналы, необходимо использовать тюнер. ТВ-тюнер преобразует сигналы аналогового и цифрового телевидения в аудио- и видеосигналы, которые могут использоваться и храниться на компьютере. Кабельные платы могут использоваться для получения телевизионных сигналов от компании кабельного телевидения. Кабельная плата требуется для доступа к кабельным каналам высококачественного вещания. Некоторые кабельные платы могут принимать до шести каналов одновременно.

  • Специализированный жесткий диск — жесткие диски с низким уровнем шума и низким энергопотреблением обычно называются аудио-видеодисками (A/V).

Вместо сборки системы HTPC некоторые клиенты могут попросить выполнить сборку домашнего сервера на базе ПК. Домашний сервер на базе ПК можно установить в любом месте дома и использовать его с нескольких устройств одновременно. Домашний сервер предоставляет общий доступ к файлам, а также выполняет потоковую передачу аудио-, видеофайлов и фотографий на компьютеры, ноутбуки, планшетные ПК, телевизоры и другие мультимедийные устройства по сети. На домашнем сервере можно настроить массив RAID для защиты ценных данных на случай сбоя жесткого диска. Для потоковой передачи данных на несколько устройств без задержек установите гигабитную сетевую интерфейсную плату.
