Системы персональных компьютеров

Внутренние компоненты ПК

Устройства хранения данных, как показано на рисунке 1, выполняют чтение информации с магнитных, оптических или полупроводниковых носителей, а также запись на них. Привод используется для постоянного хранения данных или получения информации с дискового носителя. Устройства хранения данных, например жесткий диск, могут быть установлены внутри корпуса компьютера. Для обеспечения мобильности некоторые устройства хранения данных могут быть подключены к компьютеру с помощью портов USB, FireWire, eSATA или SCSI. Эти съемные устройства хранения данных иногда называют сменными накопителями. Их можно использовать на различных компьютерах. Ниже приведены наиболее распространенные типы устройств хранения:

  • Привод гибких дисков

  • Жесткий диск

  • Привод оптических дисков

  • Флэш-накопитель

Привод гибких дисков

Привод гибких дисков, или флоппи-дисковод, — это устройство хранения данных, которое использует 3,5-дюймовые гибкие диски. На этих гибких магнитных дисках можно хранить 720 КБ или 1,44 МБ данных. В компьютере привод гибких дисков, как правило, настраивается как диск А:. Привод гибких дисков можно использовать для загрузки компьютера, если в него вставлен загрузочный гибкий диск. 5,25-дюймовые гибкие диски считаются устаревшими и практически не используются.

Жесткий диск

Жесткий диск, или накопитель на жестком диске, — это магнитное устройство, предназначенное для хранения данных. В компьютерах с ОС Windows жесткий диск, как правило, настраивается как диск C: и содержит операционную систему и приложения. Емкость жестких дисков варьируется от гигабайтов (ГБ) до терабайтов (ТБ). Скорость работы жесткого диска измеряется в оборотах в минуту. Это скорость, с которой вращается шпиндель с пластинами, на которых записаны данные. Чем выше скорость шпинделя, тем быстрее жесткий диск получает данные с пластин. Наиболее широкое распространение имеют жесткие диски со скоростью 5300, 7200, 10 000 и до 15 000 оборотов в минуту для жестких дисков на высокопроизводительных серверах. Для увеличения емкости ЗУ можно использовать несколько жестких дисков.

В традиционных жестких дисках используется технология на основе магнитных носителей. Магнитные жесткие диски оснащены двигателями, которые вращают магнитные пластины и двигают головки диска. Однако более современные твердотельные накопители (SSD) не имеют подвижных частей. Для хранения данных в них используются полупроводники. Поскольку накопители SSD не имеют двигателей и подвижных частей, они потребляют намного меньше энергии, чем магнитные жесткие диски. Энергонезависимые микросхемы флэш-памяти управляют всеми процессами хранения на накопителях SSD. Это увеличивает скорость доступа к данным, повышает надежность работы и сокращает энергопотребление. Носители SSD имеют такие же форм-факторы, что и магнитные жесткие диски, и такие же интерфейсы ATA или SATA. Магнитный жесткий диск можно заменить накопителем SSD.

Ленточный накопитель

Магнитные ленты используются в большинстве случаев для резервного копирования или архивирования данных. В ленточном накопителе используется магнитная головка чтения и записи. Скорость получения данных с ленточного накопителя может быть очень высокой, однако поиск определенных данных может занимать очень много времени, поскольку лента должна перематываться с катушки на катушку до момента нахождения данных. Наиболее широкое распространение имеют ленточные носители емкостью от нескольких гигабайт до множества терабайт.

Привод оптических дисков

В приводе оптических дисков для чтения данных с носителей используется лазер. Существует три типа приводов оптических дисков:

  • Компакт-диск (CD)

  • Универсальный цифровой диск (DVD)

  • Диск Blu-ray (BD)

Компакт-диски, диски DVD и диски Blu-ray бывают только для чтения, записываемые (однократная запись) или перезаписываемые (многократное чтение и запись). Емкость компакт-диска приблизительно равна 700 МБ. Емкость диска DVD приблизительно равна 4,7 ГБ для однослойных вариантов и около 8,5 ГБ для двухслойных. Емкость дисков Blu-ray равна 25 ГБ для однослойных вариантов и 50 ГБ для двухслойных.

Существует несколько типов оптических носителей:

  • Компакт-диск — носитель только для чтения в виде компакт-диска, на который уже записаны данные

  • Диск с однократной записью данных (CD-R) — носитель в виде компакт-диска, на который можно однократно записать данные

  • Диск с перезаписью данных (CD-RW) — носитель в виде компакт-диска, на который можно многократно записывать данные и стирать их

  • Диск DVD только для чтения — носитель только для чтения в виде диска DVD, на который уже записаны данные

  • DVD-RAM — носитель в виде диска DVD, на который можно многократно записывать данные и стирать их

  • DVD+/-R — носитель в виде диска DVD, на который можно однократно записать данные

  • DVD+/-RW — носитель в виде диска DVD, на который можно многократно записывать данные и стирать их

  • BD-ROM — носитель только для чтения в виде диска Blu-ray, на который уже записаны фильмы, игры или ПО

  • BD-R - носитель для однократной записи в формате Blu-ray, на который можно один раз записать видео с высоким разрешением (HD) и компьютерные данные

  • BD-RE - перезаписываемый носитель для многократной записи в формате Blu-ray, на который можно записать видео с высоким разрешением (HD) и компьютерные данные

Внешний флэш-накопитель

Внешний флэш-накопитель, также известный как флэшка, — это съемное устройство хранения данных, подключаемое через порт USB. Во внешнем флэш-накопителе используется тот же тип энергозависимых интегральных схем, что и в SSD. Ему не нужно электропитание для хранения данных. Операционная система получает доступ к этим устройствам хранения так же, как и к другим.

Типы интерфейсов накопителей

Жесткие диски и оптические приводы выпускаются с различными интерфейсами, которые используются для их подключения к компьютеру. Для установки привода на компьютер интерфейс подключения привода должен совпадать с типом контроллера на материнской плате. Ниже приведены распространенные интерфейсы приводов:

  • IDE — встроенный интерфейс накопителей, также называемый ATA, — ранний тип интерфейса контроллера накопителей, который служит для подключения жестких дисков к компьютеру. Для интерфейса IDE используется 40-штырьковый разъем.

  • EIDE — усовершенствованный интерфейс жестких дисков с интегрированной электроникой, также называемый ATA-2, является модернизированной версией интерфейса контроллера накопителей IDE. EIDE поддерживает жесткие диски объемом более 512 МБ, обеспечивает прямой доступ к памяти (DMA) для повышения скорости работы, а также использует интерфейс ATAPI для размещения оптических и ленточных носителей на шине EIDE. Для интерфейса EIDE используется 40-штырьковый разъем.

  • PATA — термин «параллельный интерфейс ATA» обозначает параллельный вариант интерфейса контроллера накопителей ATA.

  • SATA — термин «последовательный интерфейс ATA» обозначает последовательный интерфейс контроллера накопителей ATA. Для интерфейса SATA используется 7-контактный разъем.

  • eSATA — внешний последовательный интерфейс ATA — это внешний интерфейс для носителей SATA с возможностью «горячей замены». «Горячая замена» — это возможность подключать и отключать устройство при включенном компьютере. При использовании интерфейса eSATA внешний накопитель SATA подключается с помощью 7-контактного разъема. Длина кабеля может достигать 2 метров.

  • SCSI — интерфейс малых вычислительных систем, представляет собой интерфейс контроллера накопителя, к которому можно подключать до 15 накопителей. К интерфейсу SCSI можно подключать как внутренние, так и внешние приводы. Для интерфейса SCSI используются 25-контактные, 50-контактные или 68-контактные разъемы.

RAID — это возможность хранить данные на системе из нескольких жестких дисков. Для операционной системы RAID представляет собой один логический диск. На рисунке 2 показано сравнение различных уровней RAID. Приведенные ниже термины описывают, как RAID хранит данные на различных дисках:

  • Четность — выявление ошибок данных.

  • Чередование данных — запись данных на нескольких дисках.

  • Зеркалирование — хранение копий данных на втором диске.
