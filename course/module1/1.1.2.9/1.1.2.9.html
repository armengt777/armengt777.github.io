Системы персональных компьютеров

Внутренние компоненты ПК

Для подключения приводов необходимы шнур питания и цифровой кабель. Блок питания должен иметь разъемы питания SATA для приводов SATA, разъемы питания Molex для приводов PATA и разъемы Berg для приводов гибких дисков. Кнопки и светодиодные индикаторы, расположенные в передней части корпуса, подключаются к материнской плате с помощью кабеля передней панели.

С помощью кабелей данных приводы подключаются к контроллеру накопителей, который находится на плате адаптера или на материнской плате. Ниже приведены наиболее распространенные типы кабелей данных:

  • Кабель данных для привода гибких дисков (FDD) — имеет до двух 34-контактных разъемов для приводов и один 34-штырьковый разъем для контроллера.

  • 40-жильный кабель данных PATA (IDE/EIDE) — изначально интерфейс IDE поддерживал два устройства на один контроллер. С появлением стандарта EIDE были созданы два контроллера, способные поддерживать два устройства каждый. На 40-жильном ленточном кабеле используются 40-штырьковые разъемы. Кабель имеет 2 разъема для приводов и один разъем для контроллера.

  • 80-жильный кабель данных PATA (EIDE) — из-за роста скорости передачи данных интерфейса EIDE также возрос уровень риска повреждения данных во время передачи. Был создан 80-жильный кабель для устройств, выполняющих передачу со скоростью 33,3 МБ/с и более, который обеспечивал более надежную и сбалансированную передачу данных. На 80-жильном кабеле используются 40-штырьковые разъемы.

  • Кабель данных SATA — данный тип кабеля имеет семь проводников, один разъем с ключом для привода и один разъем с ключом для контроллера.

  • Кабель данных SCSI — существует три типа кабелей данных SCSI. В кабеле данных узкого SCSI имеется 50 проводников, до семи 50-контактных разъемов для приводов и один 50-штырьковый разъем для контроллера, также называемого управляющим адаптером. В кабеле данных широкого SCSI имеется 68 проводников, до пятнадцати 68-контактных разъемов для приводов и один 68-штырьковый разъем для управляющего адаптера. Кабель данных SCSI Alt-4 имеет 80 проводников, до пятнадцати 80-контактных разъемов для приводов и один 80-штырьковый разъем для управляющего адаптера.

ПРИМЕЧАНИЕ. Цветная маркированная жила на кабеле для привода гибких дисков или PATA обозначает контакт 1. При установке кабеля данных всегда проверяйте, соответствует ли контакт 1 кабеля контакту 1 привода или контроллера. Кабели с ключом можно подключить к приводу или контроллеру только одним способом.
