Специализированные компьютерные системы

Может потребоваться спроектировать, собрать и установить компьютеры для заказчика, выполняющего определенную задачу. Все компьютеры могут выполнять программы, хранить данные и использовать устройства ввода-вывода.

Стандартный «толстый» клиент — это традиционный настольный компьютер, отвечающий рекомендованным требованиям для Windows и выполняющий настольные приложения. Специализированный компьютер, напротив, должен поддерживать оборудование и программное обеспечение, позволяющее пользователю выполнять задачи, которые недоступны "толстому" клиенту. Примером специализированного компьютера может служить рабочая станция, используемая для компьютерного проектирования (CAD) или автоматизированного производства (CAM).

Рабочая станция CAD или CAM (CAx), как показано на рисунке, использутся для проектирования продуктов и управления процессом производства. Рабочие станции CAx используются для создания чертежей, проектирования жилых домов, автомобилей, самолетов и многих деталей товаров, вошедших в повседневный обиход. CAx используется даже для разработки частей компьютеров, используемых на рабочих станциях CAx. Компьютер, используемый для выполнения ПО CAx, должен отвечать его потребностям и потребностям устройств ввода-вывода, необходимых пользователю для проектирования и производства продукции. ПО CAx обычно является сложным и требует надежного оборудования. При необходимости в запуске ПО CAx рассмотрите варианты приобретения следующего оборудования:

  • Мощный процесссор — ПО CAx должно в очень быстром темпе выполнять чрезвычайно высокий объем вычислений. При выборе ЦП необходимо отвечать требованиям этого ПО.

  • Высокотехнологичная видеокарта — ряд ПО CAx используется для создания трехмерных моделей. Реалистичные тени и текстуры усложняют модели, поэтому необходима видеокарта, способная поддерживать высокую детализацию и высокие разрешения. Зачастую желательно или даже необходимо иметь несколько мониторов, чтобы пользователь мог работать с программным кодом, двухмерной визуализацией и трехмерными моделями одновременно. Выберите видеокарту, поддерживающую несколько мониторов.

  • ОЗУ — рабочая станция CAx обрабатывает значительные объемы данных, поэтому ОЗУ имеет очень большую важность. Чем больше ОЗУ, тем больше данных сможет обработать процессор перед чтением из более медленного хранилища, например с жестких дисков. Установите максимум памяти, поддерживаемой материнской платой и операционной системой. Объем и скорость работы памяти должны превышать минимальные требования приложения CAx.
