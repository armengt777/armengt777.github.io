Синхронизация мобильных устройств

Чтобы синхронизировать данные на устройстве iOS, нужно установить программу iTunes, как показано на рисунке. iTunes — это проигрыватель мультимедиа, который позволяет загружать, воспроизводить и упорядочивать содержимое для устройств iOS и компьютеров. iTunes также управляет устройствами iOS, активируя их в Apple и восстанавливая параметры в случае сбоя. Кроме того, iTunes применяется для обновления iOS. Ниже указаны требования к программному обеспечению для установки приложений в Windows:

  • компьютер с процессором Intel или AMD тактовой частотой 1 ГГц;

  • ОЗУ объемом 512 МБ;

  • ОС Windows XP с пакетом обновления 2 или новее, 32-разрядные версии Windows Vista или 32-разрядные версии Windows 7;

  • для 64-разрядных версий Windows Vista или Windows 7 нужна 64-разрядная программа установки iTunes.

Требования зависят от версии, а для поддержки особых возможностей аудио и видео установлены дополнительные требования. Дополнительные сведения приведены на веб-сайте Apple.

Для синхронизации содержимого на устройствах Android никакие приложения не нужны. Для автоматической синхронизации различных типов данных и содержимого с Google выберите Настройки > Личные > Учетные записи и синхронизация.

При использовании устройства Motorola с ОС Android синхронизируйте данные и содержимое с помощью Motocast USB. Ниже указаны требования к программному обеспечению для установки приложений в Windows:

  • компьютер с одноядерным процессором Intel Pentium IV либо AMD Athlon XP 2600+ или с более высокой производительностью тактовой частотой 3,0 ГГц либо компьютер с двухъядерным процессором Pentium Core Duo или с многоядерным процессором Athlon/Phenom или с более высокой производительностью тактовой частотой 1,8 ГГц;

  • 1 ГБ ОЗУ;

  • ОС Windows XP с пакетом обновления 3 или новее, Windows Vista или Windows 7.

Для поддержки особых возможностей аудио и видео могут устанавливаться дополнительные требования к оборудованию. Дополнительные сведения приведены на веб-сайте Motorola.
