Синхронизация мобильных устройств

Чтобы синхронизировать данные между устройствами, необходима общая среда связи. Наиболее часто для синхронизации данных между устройствами используются подключения USB и Wi-Fi.

Поскольку для большинства устройств Android нет программы синхронизации данных для настольных компьютеров, для синхронизации (в том числе с настольным ПК или ноутбуком) обычно используются различные веб-службы Google. Одно из преимуществ такого способа синхронизации данных заключается в том, что доступ к данным можно получить с любого компьютера или мобильного устройства в любое время, войдя в систему под своей учетной записью Google. Недостатком является то, что может быть сложно синхронизировать данные с программами, установленными на локальном компьютере, такими как Outlook для управления электронной почтой, календарем и контактами или iTunes для работы с музыкой и видео.

До iOS 5 синхронизацию можно было проводить, только подключив устройство к компьютеру при помощи кабеля USB. Теперь возможна беспроводная синхронизация данных с iTunes по Wi-Fi. Чтобы использовать синхронизацию по Wi-Fi, нужно сначала синхронизировать устройство iOS с iTunes при помощи кабеля USB. Необходимо также включить режим Синхронизировать по Wi-Fi на панели «Обзор» iTunes. После этого можно использовать синхронизацию по Wi-Fi или по кабельному подключению USB. Если устройство iOS находится в той же беспроводной сети, что и компьютер с установленной программой iTunes, и подключено к источнику питания, то синхронизация с iTunes выполняется автоматически.

Межплатформенная синхронизация данных

Часто приходится иметь дело с устройствами, работающими под управлением разных операционных систем. Синхронизация данных между разными операционными системами называется межплатформенной синхронизацией данных. Чтобы синхронизация выполнялась, необходимо установить сторонние приложения, которые синхронизируют данные устройства Android с Outlook или iTunes. iTunes можно установить на компьютере с ОС Windows или Apple — устройство Android или iOS может синхронизировать данные с любой из этих систем.

Для синхронизации между различными вычислительными платформами можно также использовать приложения. Dropbox — пример приложения, синхронизирующего различные типы данных, которое можно установить на разных платформах. Особенно полезна возможность настроить автоматическую синхронизацию фотографий, снятых камерой мобильного устройства. Сделанный снимок автоматически передается на удаленный сервер. Когда программа запускается на другом устройстве, например на планшете или настольном ПК, фотография автоматически загружается.
