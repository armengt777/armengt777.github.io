Сетевая связь и электронная почта

Настройка электронной почты

Программу для работы с электронной почтой можно установить в качестве части веб-обозревателя или как автономное приложение. Любую программу электронной почты можно использовать в Windows 7. Почта Windows Live — это программа, рекомендованная корпорацией Microsoft. Она позволяет управлять несколькими учетными записями электронной почты, календарями и контактами (рис. 1). Чтобы установить Почту Windows Live, загрузите и установите Основные компоненты Windows с веб-сайта Microsoft. Почта Windows Live входит в Основные компоненты Windows.

Для настройки учетной записи электронной почты требуется следующая информация:

  • Отображаемое имя — это может быть ваше настоящее имя, псевдоним или любое другое имя, которое вам нравится.

  • Адрес электронной почты — это адрес, на который будут приходить сообщения. Адрес электронной почты состоит из имени пользователя, за которым следует символ "@", а затем — домен сервера электронной почты (user@example.net).

  • Протокол, используемый сервером входящей почты, — разные протоколы предоставляют разные почтовые службы.

  • Имена серверов входящей и исходящей почты — эти имена предоставляет сетевой администратор или поставщик услуг доступа к Интернету.

  • Имя пользователя — используется для входа на почтовые серверы.

  • Пароль для учетной записи — пароль должен быть надежным, поскольку к почтовым учетным записям часто можно получить доступ с веб-сайтов.

В электронной почте используются следующие протоколы:

  • POP3;

  • IMAP;

  • SMTP;

  • MIME;

  • SSL.

Необходимо знать, как настроить устройство для поддержки правильного формата входящей почты. Программу почтового клиента можно настроить при помощи мастера (рис. 2).

POP3

POP3 обеспечивает получение электронной почты с удаленного сервера по подключению TCP/IP. При использовании POP3 копии электронных сообщений не остаются на сервере, однако в некоторых случаях пользователи могут настроить хранение почты в течение определенного периода. POP3 подходит для конечных пользователей с нестабильным подключением, например с коммутируемым доступом к сети. Пользователи POP3 могут подключиться к сети, загрузить электронную почту с сервера и отключиться. POP3 обычно использует порт 110.

IMAP

IMAP позволяет локальным почтовым клиентам получать электронную почту с сервера. Как и в случае POP3, при использовании IMAP можно загружать сообщения с сервера электронной почты через почтовый клиент. Различие заключается в том, что протокол IMAP позволяет пользователю упорядочивать почту на сетевом сервере электронной почты и загружать копии сообщений. Исходные сообщения остаются на сетевом сервере электронной почты. В отличие от POP3, при использовании IMAP исходные сообщения обычно хранятся на сервере, пока пользователь не переместит их в личную папку в своем почтовом приложении. IMAP синхронизирует папки электронной почты между сервером и клиентом. IMAP работает быстрее, чем POP3, но для него требуется больше места на сервере и больше ресурсов ЦП. Самая последняя версия IMAP — IMAP4. IMAP4 часто применяется в крупных сетях, например на территории учебных комплексов. IMAP обычно использует порт 143.

SMTP

SMTP — это простой текстовый протокол для передачи электронной почты по сети TCP/IP, определяющий формат электронных сообщений, которые используют только кодировку ASCII. SMTP необходим для отправки почты. SMTP обеспечивает передачу сообщений с почтового клиента на сервер электронной почты или между серверами. Сообщение передается после идентификации и проверки получателей. SMTP обычно использует порт 25.

MIME

MIME определяет расширенный формат электронной почты, включающий текст в кодировке ASCII и другие форматы, например изображения и документы текстового редактора. MIME обычно используется в сочетании с SMTP.

SSL

Протокол SSL был разработан для защищенной передачи файлов. Все данные, которыми обмениваются почтовый клиент и сервер электронной почты, зашифрованы. При настройке использования SSL на почтовом клиенте правильно укажите номер порта для сервера электронной почты.

Exchange

Exchange — это программное обеспечение Microsoft для работы почтового сервера, управления контактами и ведения календаря. Exchange использует закрытую архитектуру обмена сообщениями, которая называется интерфейсом API передачи сообщений (MAPI). При помощи MAPI приложение Microsoft Office Outlook подключается к серверам Exchange для предоставления функций управления электронной почтой, календарями и контактами.
