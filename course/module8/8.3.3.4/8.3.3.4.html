Настройка электронной почты

Для настройки устройства iOS требуется идентификатор Apple ID. Apple ID используется для доступа к магазинам Apple App Store, iTunes Store и к сервису iCloud, который предоставляет возможность работать с электронной почтой и хранить содержимое на удаленных серверах. Электронная почта в iCloud бесплатна. Пользователь получает в свое распоряжение удаленное хранилище для резервных копий, почты и документов.

Все устройства, приложения и содержимое iOS связаны с Apple ID. Во время первого включения устройства iOS помощник по настройке предоставит инструкции по подключению устройства, созданию идентификатора Apple ID и входу в систему с его использованием. Помощник по настройке также позволит создать учетную запись электронной почты в iCloud. В процессе установки можно восстановить параметры, содержимое и приложения с другого устройства iOS из резервной копии iCloud.

Если служба iCloud не была настроена в процессе установки, перейдите по следующим пунктам: Настройки > iCloud.

Чтобы настроить дополнительные учетные записи электронной почты, выполните следующие действия.

Этап 1. Выберите Настройки > Почта, адреса, календари > Добавить учетную запись.

Этап 2. Коснитесь типа учетной записи.

Этап 3. Если нужного типа учетной записи нет в списке, выберите Другое.

Этап 4. Введите сведения учетной записи.

Этап 5. Коснитесь кнопки Сохранить.
