Приложения

Работа с приложениями и папками в iOS аналогична работе в Android. Вместо кнопки «Все приложения» все приложения, установленные на устройстве, размещаются на главных экранах. Приложения можно удалить с устройства, а затем установить заново при помощи iTunes.

На каждом главном экране имеется сетка, в которой можно разместить приложения. Чтобы переместить приложение, выполните следующие действия.

Этап 1. Коснитесь приложения и удерживайте его, пока оно не начнет покачиваться.

Этап 2. Перетащите приложение на свободное место любого главного экрана.

Этап 3. Отпустите палец.

Этап 4. Переместите любые другие приложения.

Этап 5. Нажмите кнопку На главный экран, чтобы сохранить изменения.

Чтобы удалить приложение с устройства iOS, выполните следующие действия.

Этап 1. Коснитесь приложения и удерживайте его, пока оно не начнет покачиваться.

Этап 2. Коснитесь значка X на приложении.

Этап 3. Удалите любые другие приложения.

Этап 4. Нажмите кнопку На главный экран, чтобы сохранить изменения.

Панель многозадачности

В iOS можно запускать несколько приложений одновременно. Когда пользователь работает в одном приложении, другие могут выполняться в фоновом режиме. Панель многозадачности используется для быстрого переключения между недавно использовавшимися приложениями, для закрытия работающих приложений и для доступа к часто используемым настройкам. Чтобы открыть панель многозадачности, дважды нажмите кнопку На главный экран. На панели многозадачности можно выполнять следующие действия:

  • Коснуться любого приложения, чтобы открыть его.

  • Провести пальцем по панели многозадачности влево, чтобы открыть дополнительные приложения.

  • Провести пальцем по панели многозадачности вправо, чтобы открыть часто используемые настройки (рис. 1).

  • Коснуться приложения и удерживать его, пока оно не начнет покачиваться, а затем коснуться маленького красного круга с белой точкой, чтобы принудительно закрыть приложение.

Папки

Папки также можно создавать на устройствах iOS, чтобы упорядочить компоненты. Чтобы создать папку, выполните следующие действия.

Этап 1. Коснитесь приложения на главном экране и удерживайте его, пока оно не начнет покачиваться.

Этап 2. Перетащите приложение на другое приложение, которое нужно поместить в ту же папку.

Этап 3. Отпустите палец.

Этап 4. Добавьте любые другие приложения в папку.

Этап 5. Коснитесь кнопки На главный экран, чтобы сохранить изменения.

Чтобы открыть папку, коснитесь ее. Чтобы открыть приложение в папке, коснитесь его. Чтобы переименовать папку, коснитесь ее, коснитесь имени папки и введите новое имя. Чтобы закрыть папку, коснитесь экрана за пределами папки либо нажмите кнопку «На главный экран». Перемещать папки между главными экранами можно так же, как приложения.

Чтобы удалить приложение из папки, выполните следующие действия.

Этап 1. Откройте папку.

Этап 2. Коснитесь приложения, которое нужно удалить, и удерживайте его.

Этап 3. Перетащите приложение на свободное место главного экрана.

Этап 4. Отпустите палец.

Чтобы удалить папку, удалите из нее все приложения.

Многие приложения на устройствах iOS отображают оповещения (на значке приложения). Предупреждения отображаются в виде маленького образа на значке приложения (рис. 2). Число на значке показывает количество оповещений от данного приложения, которые требуют внимания пользователя. Это может быть количество пропущенных звонков, непрочитанных текстовых сообщений или доступных обновлений. Значок предупреждения с восклицательным знаком указывает на проблему с данным приложением. Если приложения расположены в папке, предупреждения отображается на значке папки. Коснитесь приложения, чтобы просмотреть оповещения.
