POST и BIOS

В BIOS содержится программа настройки параметров для оборудования. Данные настройки сохраняются в специальной микросхеме памяти, изготовленной по технологии комплементарного металло-оксидного полупроводника (КМОП или CMOS), в ИТ обычно называемой CMOS, см. рис.1. Данные в CMOS сохраняются благодаря специальной батарее в компьютере. Если батарея разряжается, все данные настройки BIOS теряются. Если это произошло, замените батарею и выполните повторную настройку тех параметров BIOS, для которых не используются значения по умолчанию.

Во многих современных материнских платах для хранения параметров BIOS применяется энергонезависимая память. Для сохранения параметров данному типу памяти питание не требуется. В таких системах батарея используется только для поддержания правильного времени и даты на часах. При разрядке или удалении батареи параметры BIOS не теряются.

Для входа в программу настройки BIOS необходимо нажать соответствующую клавишу или комбинацию клавиш во время процедуры POST. Многие материнские платы во время выполнения процедуры POST выводят изображение, называемое окно-заставка. Из-за окна-заставки компьютер может не выводить информацию о том, какую клавишу или последовательность клавиш нужно нажать. В большинстве компьютеров для входа в программу настройки BIOS требуется нажать клавишу DEL или Function. Клавишу или сочетание клавиш для конкретного компьютера можно найти в документации к материнской плате.

  • Main — базовая настройка системы

  • Advanced — дополнительные параметры системы

  • Boot — варианты загрузочных устройств и порядок загрузки

  • Security — параметры безопасности

  • Power — дополнительные параметры управления питанием

  • JUSTw00t! — дополнительные параметры напряжения и часов

  • Exit — варианты выхода из BIOS и загрузки значений по умолчанию
