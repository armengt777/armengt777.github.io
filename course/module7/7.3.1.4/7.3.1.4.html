Питание ноутбука

Параметры питания

Служебная программа управления питанием в Windows позволяет сократить потребление энергии определенными устройствами или всей системой. Управлением питанием возможно для следующих объектов:

  • Ноутбук

  • Жесткий диск

  • Монитор

  • Таймеры автоматического отключения

  • Предупреждения о низком заряде батареи

Для настройки параметров управления питанием в ОС Windows 7 и Windows Vista перейдите по следующим пунктам:

Пуск > Панель управления > Электропитание

Для настройки параметров управления питанием в Windows XP перейдите по следующим пунктам:

Пуск > Панель управления > Электропитание

Режимы питания ноутбука

Если при нажатии кнопки питания не требуется полностью отключать ноутбук, можно отрегулировать параметры, чтобы сократить энергопотребление.

Чтобы получить доступ к меню «Определение кнопок питания и включение парольной защиты» в ОС Windows 7 и Windows Vista, щелкните ссылку Действия кнопок питания в левой части окна служебной программы управления питанием. В ОС Windows XP можно открыть параметры кнопок управления питанием, щелкнув вкладку Дополнительно в служебной программе управления питанием.

В ОС Windows 7 и Windows Vista режимы следующие:

  • Действие не требуется — компьютер продолжает работу на полной мощности.

  • Сон — документы и данные приложений сохраняются в ОЗУ, что позволяет компьютеру быстро перейти в прежний режим работы.

  • Гибернация — документы и данные приложений сохраняются во временный файл на жестком диске. Из режима гибернации портативный компьютер выходит несколько дольше, чем из режима сна.

В ОС Windows XP режимы следующие:

  • Ждущий режим — документы и данные приложений сохраняются в ОЗУ, что позволяет компьютеру быстро перейти в прежний режим работы.

  • Спящий режим — документы и данные приложений сохраняются во временный файл на жестком диске. Из режима гибернации (спящего режима) ноутбук выходит несколько дольше, чем из ждущего режима.

На рисунке 1 показан режим гибернации, выбранный в служебной программе управления питанием ОС Windows 7.

Управление питанием жесткого диска и монитора

Основной объем потребления энергопитания в ноутбуке приходится на жесткий диск и монитор. Как показано на рисунке 2, можно выбрать, когда отключать жесткий диск или монитор, если ноутбук работает от аккумулятора или адаптера переменного тока.

Для настройки параметров питания жесткого диска, монитора или другого компонента компьютера в ОС Windows 7 и Windows Vista выполните следующие действия:

Этап 1. Нажмите кнопку Пуск > Панель управления > Электропитание.

Этап 2. Выберите план электропитания.

Этап 3. Щелкните пункт Настройка плана электропитания.

Этап 4. Щелкните пункт Изменить дополнительные параметры питания.

Для настройки дополнительных параметров управления питанием в Windows XP перейдите по следующим пунктам:

Пуск > Панель управления > Электропитание > Дополнительно (вкладка)

Таймеры автоматического отключения

Пользовательские параметры таймеров автоматического отключения планов электропитания в ОС Windows 7 и Windows Vista показаны на рисунке 3. Планы электропитания в ОС Windows XP показаны на рисунке 4.

Для настройки таймеров автоматического отключения в Windows 7 и Windows Vista выполните следующие действия:

Этап 1. Нажмите кнопку Пуск > Панель управления > Электропитание.

Этап 2. Щелкните пункт Настройка перехода в спящий режим и укажите необходимое время.

Для настройки параметров таймеров автоматического отключения в Windows XP перейдите по следующим пунктам:

Пуск > Панель управления > Электропитание и укажите необходимое время

Предупреждения о низком заряде аккумулятора

В ОС Windows пользователь может устанавливать уровни для предупреждений о низком заряде аккумулятора. По умолчанию предупреждение появляется тогда, когда заряд составляет 10 процентов от емкости аккумулятора. Критически низкий уровень заряда по умолчанию составляет 5 процентов. Можно также настроить тип уведомления и действие, которое следует предпринять, например перейти в спящий режим, режим гибернации или отключить компьютер, если заряд аккумулятора достигнет указанного уровня.
