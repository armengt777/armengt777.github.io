Установка и настройка принтеров

Настройка режимов и параметров по умолчанию

Некоторые принтеры оснащены панелями управления с кнопками для выбора режимов. Другие принтеры используют параметры и режимы своего драйвера. Можно настраивать параметры и режимы как для всех документов, так и для отдельного документа.

Общий метод

Общий метод относится к параметрам и режимам принтера, устанавливаемым для всех документов. Они применяются каждый раз при печати документа, если только они не отменены выбором для конкретного документа.

Чтобы изменить общую настройку принтера в Windows 7, перейдите по следующим пунктам:

Пуск > Панель управления > Устройства и принтеры >, далее следует щелкнуть принтер правой кнопкой мыши

Чтобы указать принтер, используемый по умолчанию, щелкните его правой кнопкой мыши и выберите Установить принтер по умолчанию, как показано на рисунке.

Чтобы изменить ориентацию страницы на альбомную в Windows 7, щелкните принтер правой кнопкой мыши и выберите Настройка печати. На вкладке Расположение выберите пункт альбомная из раскрывающегося меню Ориентация. Нажмите ОК.

Чтобы изменить общие настройки принтера в Windows Vista и Windows XP, перейдите по следующим пунктам:

Пуск > Панель управления > Принтеры и факсы > далее следует щелкнуть принтер правой кнопкой мыши

Чтобы указать принтер, используемый по умолчанию, щелкните его правой кнопкой мыши и выберите Установить принтер по умолчанию.

ПРИМЕЧАНИЕ. В зависимости от того, какой драйвер установлен для принтера, пункт Установить принтер по умолчанию может быть недоступен. В таком случае дважды щелкните значок принтера, откроется окно «Статус документа», в котором выберите Принтер > Установить принтер по умолчанию.

Метод «В соответствии с типом документа»

Письма, электронные таблицы и цифровые изображения относятся к тому типу документов, для которого могут потребоваться специальные параметры принтера. Изменить параметры для индивидуального документа можно с помощью изменения параметров печати документа.
