Установка и настройка принтеров

Установка и обновление драйверов устройств, микропрограммы и ОЗУ

После установки принтера распечатайте тестовую страницу, чтобы убедиться, что принтер работает правильно. Тестовая страница подтверждает, что драйвер принтера установлен и работает правильно, что принтер и компьютер взаимодействуют.

Печать тестовой страницы

Для печати тестовой страницы вручную в Windows 7 выполните следующее:

Пуск > Устройства и принтеры для вывода панели управления «Устройства и принтеры»

В Windows 7 щелкните нужный принтер правой кнопкой мыши и выполните следующее:

Свойства > Общие > Печать тестовой страницы

Для печати тестовой страницы вручную в Windows Vista выполните следующее:

Пуск > Панель управления > Принтеры для вывода меню «Принтеры и факсы»

В меню «Принтеры и факсы» щелкните нужный принтер правой кнопкой мыши и выполните следующее:

Свойства > Общие > Печать тестовой страницы

В диалоговом окне появится вопрос, правильно ли работает принтер. Если страница не напечаталась, встроенные файлы поддержки помогут устранить эту проблему.

Для печати тестовой страницы вручную в Windows XP выполните следующее:

Пуск > Принтеры и факсы для вывода меню «Принтеры и факсы». В меню «Принтеры и факсы» щелкните принтеру правой кнопкой мыши и выберите Свойства > Общие > Печать тестовой страницы

Печать из приложения

Можно проверить принтер, распечатав страницу из приложения, например Блокнот или WordPad. Для доступа к программе Блокнот в Windows 7, Windows Vista и Windows XP используйте следующий путь:

Пуск > Все программы > Стандартные > Блокнот

В открывшемся пустом документе наберите произвольный текст. Распечатайте документ, перейдя по следующим пунктам:

Файл > Печать

Проверка принтера

Можно запустить печать из командной строки, чтобы проверить принтер. Из командной строки можно печатать только файлы ASCII, например .txt и .bat.

Чтобы отправить файл на печать из командной строки в Windows 7, нажмите кнопку Пуск. В поле «Поиск программ и файлов» введите cmd и нажмите OК. В командной строке введите команду Print имя файла.txt.

Чтобы отправить файл на печать из командной строки в Windows Vista, перейдите по следующим пунктам:

Пуск > Начать поиск. В диалоговом окне Run введите cmd и нажмите OК. В командной строке введите команду Print имя файла.txt

Чтобы отправить файл на печать из командной строки в Windows XP, перейдите по следующим пунктам:

Пуск > Выполнить. В диалоговом окне Run введите cmd и нажмите OК. В появившейся командной строке введите команду: Print имя файла.txt

Проверка принтера через панель принтера

На большинстве принтеров имеется передняя панель с элементами управления, позволяющая пользователям печатать тестовые страницы. Этот метод дает возможность проверить работу принтера без подключения к сети или компьютеру. Ознакомьтесь с веб-сайтом производителя или документацией к принтеру, чтобы узнать, как напечатать тестовую страницу с помощью передней панели принтера.

Установка любого устройства не будет считаться завершенной до тех пор, пока не проверены все его функции. Принтер может иметь следующие функции:

  • печать двухсторонних документов;

  • использование разных приемных лотков для бумаги разного размера;

  • изменение параметров цветного принтера на черно-белую печать или печать в оттенках серого;

  • режим черновой печати;

  • использование приложения для оптического распознавания символов (optical character recognition — OCR).

Многофункциональное устройство может иметь следующие функции:

  • передача факса на заранее проверенный факсимильный аппарат;

  • создание копий документа;

  • сканирование документа;

  • печать документа.

ПРИМЕЧАНИЕ. Информацию об устранении замятия бумаги, установке чернильных картриджей и загрузке бумаги в лоток можно найти в прилагающейся документации или на веб-сайте производителя.
